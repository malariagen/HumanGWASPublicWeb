[@Snp] Frequencies Map

This view displays the prevalence of a [@snp] on a subcontinent scale.

Actions

  • Click the 'Select a [@snp] to display on the map' to open the 'Find [@snp]' wizard.
  • Once a [@snp] has been selected using the wizard, a summary of the active [@snp] will appear in the left-hand pane including:
    • [@Snp] name
    • [@Snp] type
    • Gene identifier
    • Gene names
    • [@Snp] identifier
    • Reference allele
    • Non-reference allele
    • Ancestral allele
    • A table displaying the allele frequencies for non-reference allele (NRAF), derived allele (DAF), minor allele (MAF)
  • Select the frequency type from the drop-down list above the map.
  • Click a pie chart to open a pop-up with more information about the selected [@snp] frequency for the selected frequency type at the selected geographic location.
  • From the info pop-up, click the 'Sample classification' button to view information about the sampling sites and samples that contributed to the relevant sample classification.
