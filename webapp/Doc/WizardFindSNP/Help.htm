'Find [@Snp]' wizard

This wizard uses several methods to search for [@snps] of interest in the [@organismshort] genome:
  • Select from a list of known [@resistmarkers]
    Select a particular [@snp] from a list of loci associated with drug resistance. The list only contains those [@snps] called in this data set. Click 'Finish' to open a pop-up with more information on the selected [@snp].
  • Search by [@snp] ID
    Each variant called in this data set is assigned a unique identifier composed of the chromosome and position (chromosome:position, e.g. MAL7:459003). To search by variant ID, enter the ID (chromosome:position) in the input box and click 'Finish'. If a [@snp] in this data set has been assigned this ID, a pop-up with more information on the selected [@snp] will appear. If this data set does not contain a [@snp] with this ID, an 'Invalid [@snp] ID' pop-up will appear.
  • Search by gene
    Search for a [@snp] in two steps:
    1. Select a gene in the [@organismshort] genome by specifying a keyword or partial keyword that is matched against the gene name and description. If the search returns a list of matches, select the gene of interest and click 'Next'.
    2. Within that gene, select a [@snp] from the list of [@snps] called in this gene. Click 'Finish' to open a pop-up with more information on the selected [@snp].
  • Search by genomic region
    Select a chromosome from the drop-down list, and specify a start and stop position in base pairs. The wizard will display a list of all [@snps] within this region. Select a specific [@snp] from this list and click 'Finish' to open a pop-up with more information on the selected [@snp].

Recent hits

This section lists the search results returned by the 'Find [@snp]' wizard during the active session. The hits are listed by [@snp] ID; click on an ID to open the info pop-up for that [@snp].

[@Snp] info pop-ups

Upon completing a successful search, an info pop-up will appear with basic information about the selected [@snp]:
  • Name
  • Type
  • Gene identifier
  • Gene names
  • Reference allele
  • Non-reference allele
  • Ancestral allele
  • Reference allele
  • Population frequencies for non-reference allele (NRAF), derived allele (DAF), minor allele (MAF)
A set of buttons leads to different views, each visualising data about the [@snp]:
  • The position on the genome
  • Map of allele frequencies
  • Info pop-up for the gene contatining this [@snp]
